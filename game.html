🎮 Interactive Learning Game

Lyra Quest

Learn the entire Lyra programming language through 15 interactive challenges. From Hello World to building a full GUI app — all inside a beautiful dark-mode game.

Download & Play How to Play →
How to Play

Your adventure starts here

📖

Story-Driven Learning

Each level has a story — you are an adventurer unlocking ancient Lyra scrolls. Each scroll teaches a new concept.

⌨️

Write Real Code

You type actual Lyra code to solve each challenge. The game checks your answer and runs it.

💡

Smart Hints

Stuck? Press Get Hint for guidance. After 3 failed attempts, a hint appears automatically.

📚

Show Example

If you're really stuck, press Show Example — but it costs a life. Study it, then try again your way.

🏆

Earn Badges

Complete each level to earn a badge — First Words, Logic Master, Golem Maker, Quill Master, and more.

📈

Progressive Difficulty

15 levels from Hello World through Variables, Loops, Functions, Classes, Files, HTTP, Database and GUI.
