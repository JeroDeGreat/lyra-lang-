v2.0.0
Lyra
Latest Stable
2025

The first full release under the name Lyra. File extension changed to .lyr and CLI command is now lyra. Major additions: AI and machine learning, professional dark-mode GUI, SQLite database, HTTP server, cryptography, networking, blockchain, and the Lyra Quest learning game. Only two symbols allowed in all code: , and .

  • Renamed from Quill to Lyra
  • File extension .lyr
  • Only , and . as symbols — enforced everywhere
  • Professional dark-mode GUI
  • Charts: bar, line, pie
  • SQLite database built in
  • AI: Naive Bayes, regression, k-NN, k-Means
  • Cryptography: hashing, tokens, passwords
  • Networking: DNS, port scan, TCP
  • HTTP server and REST client
  • Blockchain: mining and validation
  • Lyra Quest — 15-level learning game
  • 6 new stdlib modules
  • 30+ working example programs
  • Windows, Linux, macOS support
  • VS Code extension with lyre icon
⬇ lyra-lang-2.0.0.zip ~300 KB 🧩 VS Code Extension (.vsix) 📖 Install Instructions

cd "C:\Users\YourName\Downloads\lyra-lang" powershell -ExecutionPolicy Bypass -File ".\install.ps1"

Verify:

lyra version
v1.0.0
Quill
Legacy
2025

The original release — named Quill with file extension .ql and CLI command quill. Contains the core language: variables, arithmetic, conditions, loops, functions, lists, maps, classes, file I/O, error handling, and basic GUI. Fully functional for learning and small projects.

  • File extension .ql
  • CLI command: quill
  • Variables and arithmetic (English words)
  • If, else, loops (repeat, while, for each)
  • Functions with parameters and return
  • Lists, maps, string operations
  • Classes with inheritance
  • File read, write, append, delete
  • Error handling with try and catch
  • Basic tkinter GUI
  • Standard library: math, files, random, time, os, json, web
  • VS Code extension with syntax highlighting
  • Windows installer (.ps1)
⬇ quill-lang-1.0.0.zip
Note: uses quill run file.ql — not lyra

Install on Windows (v1.0.0):

cd "C:\Users\YourName\Downloads\quill-lang" powershell -ExecutionPolicy Bypass -File ".\install.ps1" quill version

How to add a new version

When you release a new version, follow these two steps to add it to this page. No coding knowledge needed — just copy and paste.

Step 1 — Upload your new zip file to GitHub

Go to your GitHub repo → Add file → Upload files → drag your new lyra-lang-X.X.X.zip into the downloads/ folder → Commit.

Step 2 — Copy this block into versions.html

Open versions.html on GitHub, click the pencil icon to edit, then paste this block above the v2.0.0 section and fill in the details:

<!-- PASTE THIS BLOCK FOR EACH NEW VERSION --> <div class="release latest"> <div class="release-header"> <div class="release-version">v3.0.0</div> <div class="release-name">Lyra — The Compiler</div> <span class="tag tag-latest">Latest</span> <span class="tag tag-stable">Stable</span> <div class="release-date">2026</div> </div> <p class="release-desc"> Write your description here. </p> <ul class="feature-list"> <li>New feature one</li> <li>New feature two</li> <li>New feature three</li> </ul> <div class="download-row"> <a href="downloads/lyra-lang-3.0.0.zip" class="dl-btn dl-btn-primary" download> ⬇ lyra-lang-3.0.0.zip </a> <a href="downloads/lyra-lang.vsix" class="dl-btn dl-btn-ghost" download> 🧩 VS Code Extension </a> </div> </div> <!-- END NEW VERSION BLOCK -->

Also change the old "Latest" card to use class="release stable" and change its tag from tag-latest to tag-stable.
