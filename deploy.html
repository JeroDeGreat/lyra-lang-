🚀 Free Hosting Guide

Get Your Website Live

Host your Lyra website on GitHub Pages — completely free, forever. Takes about 10 minutes start to finish.

1

Create a free GitHub account

If you already have one, skip to step 2.

  • Go to github.com and click Sign up
  • Enter your email address and create a password
  • Choose a username — this becomes part of your URL, so pick something you like. Something like jerowgrabo or lyra-lang works great
  • Verify your email when GitHub sends the confirmation
2

Create a new repository

A repository is just a folder on GitHub that holds your files.

  • Once logged in, click the + icon in the top right corner
  • Click New repository
  • Repository name: type lyra-lang
  • Description: Lyra — Write code the way you think.
  • Make sure Public is selected — required for free hosting
  • Tick Add a README file
  • Click the green Create repository button
3

Upload the website files

Extract lyra-website.zip on your computer first. Open the extracted folder — you will see index.html, assets/, downloads/, and other files.

  • On your new GitHub repo page, click Add file → Upload files
  • Open the extracted website folder in File Explorer
  • Press Ctrl+A to select everything inside it
  • Drag all the files and folders into the GitHub upload box in your browser
  • Wait for all files to show green checkmarks
  • At the bottom, in the commit message box type: Add Lyra website
  • Click Commit changes
⚠ Make sure you upload the contents of the website folder, not the folder itself. GitHub should show index.html at the root of your repo, not inside a subfolder.
4

Upload your download files

This makes the download buttons on the website actually work. You need to upload your zip files into the downloads/ folder.

  • On your repo, click the downloads folder
  • Click Add file → Upload files
  • Drag in these files from your computer:
    • lyra-lang-2.0.0.zip — the current Lyra release
    • lyra-lang.vsix — the VS Code extension
    • quill-lang-1.0.0.zip — the original v1.0.0 release (find this in your Downloads folder from the first session)
  • Click Commit changes
5

Turn on GitHub Pages

This is the step that makes your site go live.

  • On your repo page, click the Settings tab at the top
  • In the left sidebar, scroll down and click Pages
  • Under Source, click the dropdown showing None
  • Select Deploy from a branch
  • Under Branch, select main
  • Make sure the folder shows / (root)
  • Click Save
✓ After saving, GitHub will show a message at the top of the Pages settings:
"Your site is live at https://YOUR-USERNAME.github.io/lyra-lang/"
It takes 1–2 minutes to go live. If you see a 404, wait a moment and refresh.
6

Your site is live

Visit your site at:

https://YOUR-USERNAME.github.io/lyra-lang/

Replace YOUR-USERNAME with the GitHub username you created in Step 1. For example if your username is jerowgrabo, your site is at:

https://jerowgrabo.github.io/lyra-lang/

Share this link — anyone in the world can visit it and download Lyra.

7

Get a custom domain — optional but free to set up

If you want lyra-lang.org instead of the long GitHub URL, you need to buy a domain. This is the only part that costs money — about $10–15 per year. Namecheap.com is a good place to buy one.

  • Buy your domain at namecheap.com
  • In Namecheap, go to Domain List → click Manage next to your domain → click Advanced DNS
  • Add a new record:
    • Type: CNAME
    • Host: @
    • Value: YOUR-USERNAME.github.io
  • Back on GitHub → Settings → Pages → under Custom domain, type your domain (e.g. lyra-lang.org) → click Save
  • Tick Enforce HTTPS — GitHub gives you a free SSL certificate
  • Wait 5–30 minutes for DNS to update
✓ After DNS updates, anyone can visit lyra-lang.org and get the full site with HTTPS — at no ongoing cost beyond the domain renewal.
8

Update the site in the future

Whenever you want to change something — add a new version, update text, fix a typo:

  • Go to your GitHub repo at github.com/YOUR-USERNAME/lyra-lang
  • Click the file you want to change (e.g. versions.html)
  • Click the pencil icon (Edit) in the top right
  • Make your changes
  • Click Commit changes at the bottom
  • Your site updates within about 60 seconds

For uploading new files (like a new version zip), click Add file → Upload files inside the downloads/ folder.
