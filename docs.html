Quick Reference

The two symbols

The comma ,

note Separates items and arguments.

let x, y be 1, 2.
let fruits be a list of
  "apple", "banana", "cherry".
define add taking a, b.
  return a plus b.
end define.

The period .

note Ends every statement, like
note a period ends a sentence.

let name be "Alice".
say "Hello, {name}!".
if name is "Alice".
  say "Hi Alice!".
end if.

String Interpolation

let name be "Alice".
let age be 30.
say "Hello, {name}!".
say "You are {age} years old.".
say "List has {the size of fruits} items.".

Comments

note This is a comment.

begin note.
  This spans
  multiple lines.
end note.
All Modules

Standard Library

use math.

say pi.
say e.
let r be square root of 144.

use files.

write "Hello" to file "out.txt".
let c be the content of file "out.txt".
delete file "out.txt".

use random.

let n be random number
  between 1 and 100.

use data.

let rows be readcsv with "data.csv".
let avg be average with numbers.
let sorted be sortrows with rows, "name".

use ml.

let model be trainbayes
  with emails, labels.
let r be predictbayes
  with model, "new text".

use crypto.

let h be hash with "password".
let t be token.
let s be passwordstrength with "pwd".

use net.

let ip be dnslookup with "google.com".
let open be isportopen with "host", 80.

use blockchain.

let r be newblockchain with "chain".
let h be addblock with "chain", "tx".
let v be validateblockchain with "chain".
