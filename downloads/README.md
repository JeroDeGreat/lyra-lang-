# Downloads

Put your release files in this folder so the website download buttons work.

## Files needed

| File | What it is |
|---|---|
| `lyra-lang-2.0.0.zip` | Lyra v2.0.0 — current release |
| `lyra-lang.vsix` | VS Code extension — current version |
| `quill-lang-1.0.0.zip` | Quill v1.0.0 — original legacy release |

## How to add files

1. Go to your GitHub repo
2. Click `downloads/` folder
3. Click **Add file** → **Upload files**
4. Drag in your zip or vsix files
5. Click **Commit changes**

The download buttons on the website will immediately start working.
